# コミックビューアー

ローカルでマンガを閲覧できるWebアプリケーションです。

## 機能

- マンガ一覧表示
- タグによる検索・フィルタリング
- マンガビューアー（キーボードナビゲーション対応）
- アップロード機能（ZIP/PDF対応）
- 設定画面（ページ送り方向、テーマ等）

## セットアップ

### 必要な環境
- Node.js (v16以上)
- npm

### インストール
```bash
npm install
```

## 開発・実行コマンド

### 開発モード
```bash
# 開発サーバー起動（サーバー + クライアント）
npm run dev

# サーバーのみ開発モード
npm run start:dev
```

**アクセス方法:**
- サーバー（API）: http://localhost:3000
- クライアント（UI）: http://localhost:5173

### 本番モード
```bash
# ビルド + 起動
npm run preview

# または個別に実行
npm run build
npm start
```

**アクセス方法:**
- 統合サーバー: http://localhost:3000

### その他のコマンド
```bash
# クリーンアップ（ビルドファイル削除）
npm run clean

# サーバーのみビルド
npm run build:server

# クライアントのみビルド
npm run build:client
```

## 使用方法

1. **データディレクトリの設定**
   - `server/config/config.json`でデータディレクトリを指定
   - デフォルト: `K:\Document\Comic\Data`

2. **マンガの追加**
   - アップロード画面からZIP/PDFファイルをアップロード
   - または、データディレクトリに直接フォルダを配置

3. **閲覧**
   - マンガ一覧から作品を選択
   - キーボードショートカットでページ送り

## キーボードショートカット

- **右矢印 / D**: 次のページ
- **左矢印 / A**: 前のページ
- **設定画面で方向を変更可能**

## 設定

設定画面で以下の項目を変更できます：
- ページ送り方向
- テーマ（ダーク/ライト/自動）
- 1ページあたりの表示件数
- 設定のエクスポート/インポート

## プロジェクト構成

```
Viewer/
├── client/          # フロントエンド (Vite + TypeScript)
├── server/          # バックエンド (Express + TypeScript)
├── shared/          # 共通型定義
└── package.json     # ルート設定
```
